## UI Prompt Appendix

Add the following section near the end of your prompt so the AI updates the existing UI instead of redesigning everything.

```text
==================================================
BRANDING & COLOR SYSTEM (MANDATORY)
==================================================

Implement the following color palette consistently throughout the application. Do NOT introduce additional primary colors.

Primary Brand Color
#8EB69B

Secondary Background Color
#DAF1DE

Rules

• Primary color (#8EB69B) should be used for:
  - Primary buttons
  - Active navigation items
  - Icons
  - Charts
  - Status highlights
  - Badges
  - Links
  - Focus rings
  - Progress bars
  - Toggle switches
  - Section accents

• Background color (#DAF1DE) should be used for:
  - Main application background
  - Dashboard background
  - Page sections
  - Cards where appropriate
  - Empty states
  - Hover surfaces
  - Forms
  - Tables
  - Dialog backgrounds

• The UI should feel soft, calming, premium, and healthcare-oriented with an Ayurvedic aesthetic.

• The application should primarily use only these two brand colors, along with neutral whites, blacks, and grays for readability.

• Avoid bright blue, purple, orange, or red as theme colors except where required for warning/error states.

==================================================
COMPONENT STYLING
==================================================

Update all existing components to follow the new branding without changing functionality.

Apply the color palette to:

• Sidebar
• Top Navigation
• Dashboard Cards
• Statistics Cards
• Buttons
• Tables
• Forms
• Input Fields
• Dropdowns
• Search Bars
• Tabs
• Badges
• Notifications
• Modals
• Tooltips
• Pagination
• Charts
• Calendar
• Appointment Cards
• Patient Cards
• Medicine Cards
• Package Cards
• Billing Screens
• Inventory Screens
• Reports
• Admin Module

==================================================
DARK MODE
==================================================

Keep Dark Mode fully functional.

Dark Mode should use darker neutral backgrounds while preserving the brand identity by using:

Primary Accent
#8EB69B

Hover/Selected States
Use lighter or darker shades derived from #8EB69B.

==================================================
DESIGN GOAL
==================================================

The final UI should resemble a modern premium healthcare application with a calm Ayurvedic theme. Maintain consistency across every screen, avoid unnecessary gradients, excessive shadows, or flashy animations, and prioritize a clean, minimal, professional interface.
```
